This is critical — when users share your link on WhatsApp,
it must show a rich preview card. Add to every movie page:
<meta property="og:type" content="video.movie" />
<meta property="og:title" content="{Movie} — {OTT} pe Dekho" />
<meta property="og:description" content="{Movie} ab {OTT} pe available hai!" />
<meta property="og:image" content="https://image.tmdb.org/t/p/w1280{backdrop}" />
<meta property="og:image:width" content="1280" />
<meta property="og:image:height" content="720" />
<meta property="og:url" content="https://filmdhundo.com/movies/{slug}" />
<meta property="og:site_name" content="FilmDhundo" />
WhatsApp uses og:image to show preview — backdrop image (1280px) works best.
