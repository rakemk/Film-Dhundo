Build a full-stack movie discovery web app called "FilmDhundo" using the following stack and requirements. Do not ask questions — build everything completely.

=== TECH STACK ===
- Frontend: HTML, CSS, Vanilla JavaScript (no frameworks)
- Backend: Node.js with Express.js
- Database: Firebase Firestore (free tier)
- Hosting-ready: Vercel / Replit deployment
- APIs: TMDB API (free), YouTube Data API (free)
- Payments: Razorpay (Indian payment gateway)
- Analytics: Google Analytics 4 (gtag.js)

=== BACKEND: server/index.js ===
Express server with these API routes:

GET /api/movies/trending
- Fetch from TMDB /trending/movie/week
- Add OTT platform mapping (assign platform based on movie id)
- Cache response for 1 hour
- Return: id, title, poster_path, vote_average, release_date, ott_platform, slug

GET /api/movies/search?q={query}
- Fetch from TMDB /search/movie with language=hi-IN
- Return top 20 results with same fields

GET /api/movies/{id}
- Fetch full movie details from TMDB
- Fetch YouTube trailer via YouTube API
- Return all details including cast, similar movies, trailer URL

GET /api/seo/sitemap
- Generate sitemap.xml dynamically
- Include all movie pages, list pages, homepage
- Return valid XML

POST /api/payment/create-order
- Create Razorpay order with amount in paise
- Return order_id, key_id

POST /api/payment/verify
- Verify Razorpay payment signature using crypto hmac
- On success, update Firestore: users/{uid}/premium = true, expiry = Date.now() + 30days
- Return success/failure status

POST /api/users/watchlist
- Save movie to Firestore watchlist for authenticated user
- Check if user is premium for unlimited watchlist
- Free users: reject if watchlist.length >= 5

=== SEO REQUIREMENTS (CRITICAL) ===
Every movie page must have:
1. Unique <title> tag under 60 characters
2. Unique <meta description> under 160 characters
3. JSON-LD schema:
{
  "@context": "https://schema.org",
  "@type": "Movie",
  "name": "{title}",
  "datePublished": "{year}",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "{rating}",
    "bestRating": "10"
  }
}
4. Open Graph tags for social sharing
5. Canonical URL
6. robots.txt allowing all pages
7. sitemap.xml with all movie URLs
